# Autonomous Runner Prompt

Run the GitHub Money Ops pipeline from the repository root.

1. Run `./scripts/run_scout.sh`.
2. Read the newest report in `reports/` and the JSON in `data/opportunities.json`.
3. Inspect the top opportunities that look lawful, active, and realistically solvable.
4. Prefer opportunities matching AI agents, TypeScript, Python automation, docs, and README improvements.
5. If a task is suitable, clone the target repository into a temporary work area, inspect the issue, run the relevant tests, and prepare a minimal patch locally.
6. Do not perform unauthorized security testing.
7. Do not spam comments or submit low-quality PRs.
8. Do not handle bank, tax, payout, wallet, or payment setup.
9. Report the best opportunity, the evidence it is worth pursuing, and the concrete next autonomous action.
